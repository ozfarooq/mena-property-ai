// n8n upgrade trigger
At 2 clients, n8n Cloud Starter ($20/mo) is the right call — no server to manage, automatic updates, 2,500 executions covers your volume comfortably.
// whatsapp free tier status
At 300 inquiries, you're well inside Meta's free 1,000 conversations/month. No WhatsApp cost until you exceed that.
Use Haiku for extraction, Sonnet for generation. Lead qualification (extracting intent, budget, location via tool use) is a simple structured task — Haiku 4.5 at $1/$5 per million tokens handles it perfectly at 5× less cost. Only call Sonnet ($3/$15) when generating the personalised WhatsApp reply where language quality matters. This split cuts your Claude bill by ~60% vs using Sonnet for everything.
Enable prompt caching for your system prompt. Your real estate system prompt (RERA rules, community guides, RAG chunks) is repeated on every API call. Anthropic's prompt caching charges only 10% of the base input price on cache hits after the first call. At 300 conversations with a 2,000-token system prompt, this alone saves ~$0.50/month — small now, meaningful at 2,000+ inquiries.
Self-host n8n on a VPS when you hit 5 clients. n8n Community Edition is 100% free with unlimited executions. A Hetzner or DigitalOcean VPS costs $6–7/month. That saves $13–14/month vs the Cloud Starter plan and removes the 2,500 execution cap entirely. The only cost is 2 hours of setup time. Upgrade trigger: 5+ brokerage clients or 1,000+ monthly executions.
